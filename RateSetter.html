Segment Expenses = fuel cost + maintenance cost (per-mile operating cost).

Empty-Mile Rates are an additional $/mile charge for non-revenue segments (deadhead, hub travel, home travel). These represent opportunity cost and are shown separately from operating expenses, but included in profit calculations.

Base Rate = $150 − ($10 × full drive hours) for route only. Decreases on longer hauls.

Optimum Route Value (ORV) = base rate + all segment expenses + $35/hr route drive time. The CPM tier closest to ORV is highlighted green.

Profit = (CPM × loaded miles) − (segment expenses + applicable mileage rates).

HOS Warning: shown if drive time exceeds 15h AND available buffer per 24h period is less than 9h. Enter Delivery Window on the Calc tab to enable this check.

Hub geocoding: Default hub coordinates are built in. If you change a hub ZIP, tap Save Settings — new ZIPs are geocoded at <1 req/sec to respect Nominatim limits.
